for initializing Git Repository
git init

==================================================
Daily use:-

For adding files in staging area
git add "File"
git add .

For Git Commit
git commit -m "Message"

For Checking Statu of the files
git status

for pushing Code to Repository
git push -u origin master
====================================================
for Creating or Switching Branches
git checkout -b "Branch Name"

for checking branch name 
git branch

for adding Local code to Remote  Repository
git remote add origin "Repository Address"
